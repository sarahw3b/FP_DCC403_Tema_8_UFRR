# FP_DCC403_Tema_8_UFRR
Este Projeto Final de Sistemas Operacionais, apresenta o FormalRace-Checker, uma ferramenta de análise estática que realiza o parsing de código-fonte C concorrente, modela as propriedades de concorrência como fórmulas proposicionais booleanas em CNF, submete-as ao solver MiniSAT e gera relatórios de auditoria estruturados com contraexemplos.
